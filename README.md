# Elm

## Start project

elm init

## Set up testing

npm install -g elm-test
elm-test init

To learn more about testing in elm: https://elmprogramming.com/easy-to-test.html

## Development environment

elm reactor

## Installing 3th party libs

elm install timjs/elm-collage
